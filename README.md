# LIF_MomentClosure
Comparison of direct Fokker-Planck integration and moment closure ODE system for Leaky IF neurons poulation.
For the population Fokker-Planck equation integration the scripts make use of NeuralFields.jl (https://github.com/giav1n/NeuralFields.jl) while for the differential equations interations we use standard tool from the Julia library DifferentialEquations.jl.
